## Syscalls

| Name          | Number | Description                                   | Requires         |
|---------------|--------|-----------------------------------------------|------------------|
| EXIT          | 0x00   | Terminate current process                     | -                |
| SPAWN         | 0x01   | Spawn new process (inherits caps from parent) | CAP_FS_READ      |
| CAP_SPAWN     | 0x02   | Spawn caps for the process                    | CAP_CAPS_MGMT    |
| SLEEP         | 0x03   | Sleep for N milliseconds                      | -                |
| OPEN          | 0x10   | Open or create a file (with O_CREAT flag)     | CAP_FS_READ      |
| CLOSE         | 0x11   | Close a file descriptor                       | -                |
| READ          | 0x12   | Read from file descriptor                     | CAP_FS_READ      |
| WRITE         | 0x13   | Write to file descriptor                      | CAP_FS_WRITE     |
| MKDIR         | 0x14   | Create a directory                            | CAP_FS_CREATE    |
| REMOVE        | 0x15   | Remove file or empty directory                | CAP_FS_DELETE    |
| DUP2          | 0x16   | Duplicate file descriptor (redirect)          | -                |
| SBRK          | 0x20   | Increase/decrease program break (heap)        | -                |
| AWAIT         | 0x30   | Wait for child process to change state        | -                |
